Remotely Controlled Home

The objective of “Remotely Controlled Home” is to establish a collaboration between home and technology to make daily chaos simpler and easier. Through this project we want to design a home automation system using, Internet of Things and Possibly Machine Learning with an added facility of mobile or web application. Remotely Controlled Home is a model implementation of home automation system with advanced features and technologies. Home automation is building automation for a home, called a smart home or smart house. A home automation system will control lighting, climate, entertainment systems, and appliances. It may also include home security such as access control and alarm systems.
